# Quick Autofill ⚡

A lightweight Chrome extension that saves your profile information once and autofills web forms instantly.

## Features

- Save personal profile data
- Autofill forms with one click
- Inline ⚡ Autofill button near form fields
- Clean and modern popup UI
- Fast and lightweight
- Built with Chrome Manifest V3

## How it works

Quick Autofill lets you store your profile information like:

- Name
- Email
- Phone
- Address

Then you can autofill forms by:

- Clicking **Autofill Page** in the extension popup
- Using the **⚡ Autofill** button that appears near form fields

## Installation

1. Clone the repository

```bash
git clone https://github.com/mashhood-ma/quick-autofill-extension.git
```

2. Open Chrome and go to:

chrome://extensions

3. Enable **Developer Mode**

4. Click **Load Unpacked**

5. Select the extension folder
